### 3-Gate Review + 4 Pitfall Checks (analyses 準拠)

出典: `wiki/analyses/dsr-protocol-incentive-operations-guide-2026-04-20.md`

  - Tail risk / Custody / Incentive 喪失 / Sybil / Hedge omission (4-Pitfall #2 Execution サブ項目)

  - サンプル事前宣言 / 比較可能性 / 層別閾値 / 反証条件 / 事後評価ループ
- **4 Pitfall**: Regime / Execution (Hedge omission 含む) / Leakage / Tail
