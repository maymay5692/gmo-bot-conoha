## 戦略評価フレームワーク (ナレッジ活用)

### 3-Gate Review + 4 Pitfall Checks (analyses 準拠)

出典: `wiki/analyses/dsr-protocol-incentive-operations-guide-2026-04-20.md`

- **Gate 1 Accountability**: Sharpe ≥ 0.5, DSR ≥ 0.95 (N_trials シナリオ全て PASS)
  - N_trials は **universe 事前客観宣言時のみ** パラメータ候補数のみで OK
  - 事後選定なら `log2(prior/final) + 1` 分上乗せ
- **Gate 2 Tail Safety**: 標準 5 項目 + 戦略固有拡張
  - Tail risk / Custody / Incentive 喪失 / Sybil / Hedge omission (4-Pitfall #2 Execution サブ項目)
- **Gate 3 OOS 再現性**: 時系列 IS/OOS が不可能な単一イベント戦略は **クロスセクショナル OOS 事前宣言式 5 要件** を適用
  - サンプル事前宣言 / 比較可能性 / 層別閾値 / 反証条件 / 事後評価ループ
- **4 Pitfall**: Regime / Execution (Hedge omission 含む) / Leakage / Tail

### Protocol Incentive Arbitrage
